# Lijing Cui — IWU GitHub Pages Site

## How to publish with GitHub Pages

1. Commit these files to the repo root (or to a `gh-pages` branch).
2. In GitHub, go to **Settings → Pages**.
3. Under **Build and deployment**, set:
   - **Source**: Deploy from a branch
   - **Branch**: `main` (root) *or* `gh-pages` (root)
4. Save. Your site will be live at:
   - `https://USERNAME.github.io/Lijing-Cui--IWU` (replace USERNAME).
